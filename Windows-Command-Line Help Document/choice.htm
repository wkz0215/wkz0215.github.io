choice

  • 项目
  • 09/21/2022

本文内容

提示用户从批处理程序中的单字符选项列表中选择一项，然后返回所选选项的索引。 如果不带参数使用， 则选择 将显示默认选项 Y 和 N。

语法

choice [/c [<choice1><choice2><…>]] [/n] [/cs] [/t <timeout> /d <choice>] [/m <text>]

参数

参数 说明
/c <choice1><choice2><…> 指定要创建的选项列表。 有效选项包括 a-z、A-Z、0-9 和扩展 ASCII 字符 (128-254) 。 默认列表为 YN，显示为 [Y,N]?。
/n 隐藏选项列表，尽管选项仍处于启用状态，并且消息文本 (如果 由 /m 指定) 仍显示。
/cs 指定选项区分大小写。 默认情况下，这些选项不区分大小写。
/t <timeout> 指定在使用 /d 指定的默认选项之前要暂停的 秒数。 可接受的值为 0 到 9999。 如果 /t 设置为 0， 则选项 在返回默认选项之前不会暂停。
/d <choice> 指定在等待 /t 指定的秒数后使用 的默认选项。 默认选项必须包含 /c 指定的选项 列表中。
/m <text> 指定在选项列表之前显示的消息。 如果 未指定 /m ，则只显示选择提示。
/? 在命令提示符下显示帮助。

注解

  • ERRORLEVEL 环境变量设置为用户从选项列表中选择的键的索引。 列表中的第一个选择返回值 ， 1第二个返回 值 2，以此类比。 如果用户按下的键不是有效的选择， 则选择 会发出警告提示音。

  • 如果 选择 检测到错误条件，则返回 ERRORLEVEL 值 。 如果用户按 CTRL+BREAK 或 CTRL+C， 则选择 将返回 ERRORLEVEL 值 。

注意

在批处理 程序中使用 ERRORLEVEL 值时，必须按降序列出这些值。

示例

若要显示选项 Y、 N 和 C，在批处理文件中键入以下行：

choice /c ync

当批处理文件运行 choice 命令时，将显示 以下 提示：

[Y,N,C]?

若要隐藏选项 Y、N 和 C，但显示文本"是"、"否"或"继续"，在批处理文件中键入以下行：

choice /c ync /n /m "Yes, No, or Continue?"

注意

如果使用 /n 参数 ，但不使用 /m，则当选择正在等待输入 时，不会 提示用户。

若要同时显示前面示例中使用的文本和选项，在批处理文件中键入以下行：

choice /c ync /m "Yes, No, or Continue"

若要将时间限制设置为 5 秒，并指定 N 作为默认值，请键入批处理文件中以下行：

choice /c ync /t 5 /d n

注意

此示例中，如果用户在五秒钟内未按键，则选择默认选择 N，并返回错误值 。 否则 ，choice 将返回与用户选择对应的值。
