diskcopy

  • 项目
  • 09/21/2022

本文内容

将源驱动器中软盘的内容复制到目标驱动器中的格式化或未格式化软盘。 如果不带参数使用 ，diskcopy 将当前驱动器用于源磁盘和目标磁盘。

语法

diskcopy [<drive1>: [<drive2>:]] [/v]

参数

参数 说明
<drive1> 指定包含源磁盘的驱动器。
/v 验证是否正确复制了信息。 此选项会降低复制过程的速度。
/? 在命令提示符下显示帮助。

注解

  • Diskcopy 仅适用于软盘等可移动磁盘，这些磁盘的类型必须相同。 不能对 硬盘使用 diskcopy 。 如果为 drive1 或 drive2 指定硬盘驱动器， diskcopy 将显示以下错误消息：

    Invalid drive specification
Specified drive does not exist or is nonremovable

    diskcopy 命令会提示你插入源磁盘和目标磁盘，并等待你在继续操作之前按键盘上的任何键。

    复制磁盘后， diskcopy 将显示以下消息：

    Copy another diskette (Y/N)?

    如果按 Y， diskcopy 会提示你插入源磁盘和目标磁盘，以执行下一个复制操作。 若要停止 diskcopy 进程 ，请按 N。

    如果要复制到 drive2 中的未格式化软盘， diskcopy 将格式化磁盘，其每个轨迹的边数和扇区数与驱动器 1 中的磁盘上相同。 Diskcopy 在格式化磁盘并复制文件时显示以下消息：

    Formatting while copying

  • 如果源磁盘具有卷序列号， diskcopy 将创建目标磁盘的新卷序列号，并显示复制操作完成后的编号。

  • 如果省略 drive2 参数， diskcopy 将使用当前驱动器作为目标驱动器。 如果省略这两个驱动器参数， diskcopy 将同时使用当前驱动器。 如果当前驱动器与 drive1 相同， diskcopy 会提示你在必要时交换磁盘。

  • 从软盘驱动器（例如 C 驱动器）外的其他驱动器运行 diskcopy 。 如果软盘 驱动器 1 和软盘 驱动器 2 相同， diskcopy 会提示你切换磁盘。 如果磁盘包含的信息多于可用内存可以容纳的信息， diskcopy 无法一次读取所有信息。 Diskcopy 从源磁盘读取数据，写入目标磁盘，并提示再次插入源磁盘。 此过程将一直进行，直到复制整个磁盘。

  • 碎片是磁盘上现有文件之间存在少量未使用的磁盘空间。 分片源磁盘可能会减慢查找、读取或写入文件的过程。

    由于 diskcopy 在目标磁盘上创建源磁盘的精确副本，因此源磁盘上的任何碎片都传输到目标磁盘。 若要避免将碎片从一个磁盘传输到另一个磁盘，请使用 copy 命令 或 xcopy 命令 复制磁盘。 由于复制和 xcopy 按顺序复制文件，因此新磁盘不会碎片化。

    注意

    不能使用 xcopy 复制启动磁盘。

  • diskcopy 退出代码：

    退出代码 说明
    0 复制操作成功
    1 发生非主要读/写错误
    3 发生严重硬错误
    4 发生初始化错误

    若要处理 diskcomp 返回的退出代码，可以在批处理程序中的 if 命令行上使用 ERRORLEVEL 环境变量。

示例

若要将驱动器 B 中的磁盘复制到驱动器 A 中的磁盘，请键入：

diskcopy b: a:

若要使用软盘驱动器 A 将一个软盘复制到另一个软盘，请首先切换到 C 驱动器，然后键入：

diskcopy a: a:
